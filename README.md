# Simple Soundboard

A lightweight, Fantasy+-style soundboard for Foundry VTT, built entirely on
top of Foundry's own Playlist/Sound system. Tag the sounds you've already
imported, trigger them from a one-click button grid instead of hunting
through the Playlists sidebar, and save/recall named scene presets (which
tracks were playing, at what volume).

This module does not include any audio — it organizes and triggers the
music/SFX you import into Foundry normally.
